# 📱 Duolingo UI Clone – Streak System

## 📌 Project Overview
This project is a front-end case study inspired by the Duolingo application.  
It replicates the **Streak System UI** and demonstrates how user engagement is increased using gamification techniques.

---

## 🎯 Objective
To understand and implement the working of:
- Streak tracking
- Weekly progress visualization
- User interaction and dynamic UI updates

---

## ✨ Features

- 🔥 **Streak Counter**
  - Increases when user clicks "Start Lesson"

- 📅 **Weekly Progress System**
  - Days (Mon–Sun) activate one by one

- 🔄 **Weekly Reset Logic**
  - After completing 7 days, the UI resets for a new week

- 📊 **Dynamic Progress Bar**
  - Updates based on number of completed days

- 💾 **Data Persistence**
  - Streak and progress are saved using localStorage

- 🎯 **Learning Path UI**
  - Displays completed, current, and locked levels
